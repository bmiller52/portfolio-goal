Here's a full bio about me

Bio

In early 2015, I discovered 3D modeling and from there I found Blender.
I immediately found the love for design and learned from the best like
Andrew Price, CG Geek, Gleb Alexandrov and many more. I can make
wallpapers, architecture, cities, future, and landscapes. My main focus is
photo realism and I strive for every render to be that way. One of my favorite
things about 3D modeling is that you can create anything. This is where my
mind runs wild with ideas.

Other Things about me

My other passions are programming and 3D printing. I started learning how to code
when I was ten years old. I first learned HTML and CSS through Khan Academy then
I took some courses with Code School. I also learned Python and made some games
with my free time. I've also made dynamic websites using Django.

Taking my skills in Blender, I am also able to engineer custom models for 3D printing.
